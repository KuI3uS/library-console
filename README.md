# Library Console

Console-based Library Management System written in Java.

Project created for learning **object-oriented programming**, clean code
and backend fundamentals step by step.

---

## Project Goal

The goal of this project is to build a simple but well-structured
library management system that allows:

- adding books
- listing all books
- searching books
- borrowing and returning books
- saving data to a file

The project is developed incrementally, following real backend
development workflow.
